# Proyecto IV


Repositorio para la asignatura Infraestructura Virtual del Grado en Ingeniería Informática de la UGR.



## *DictionaryBOT*



Se trata de un Bot de telegram que muestra si una palabra existe junto con su significado según la RAE.



**-Descripción**



El bot estará online durante todo el tiempo, de forma que irá recopilando información de manera autómata.

Uso de un bot de Telegram para  saber el significado de una palabra determinada.



**-Objetivo**



Mostrar de forma simple y sencilla el significado de una palabra cualquiera que queramos, siempre y cuando esta sea reconocida por la RAE. 



**-Herramientas**



-  API de Telegram. 

- Lenguaje Python 3.

- Servicio en la nube (Azure o Amazon).

- Entorno de pruebas C9.io
